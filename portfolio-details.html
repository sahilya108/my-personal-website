Project Overview

Stone Paper Scissors is a simple and interactive web-based game developed using HTML, CSS, and JavaScript. The game allows users to play against the computer by selecting one of the three options: stone, paper, or scissors.

The system generates a random choice for the computer and displays the result instantly. This project focuses on improving logical thinking, user interaction, and basic game development concepts while providing an enjoyable user experience.

Key Features

Responsive Design

The game interface adapts smoothly to different screen sizes, providing a consistent experience on desktop, tablet, and mobile devices.

Fair Random Gameplay

The computer’s choice is generated randomly to ensure fair and unpredictable game results every time.

Instant Result Display

Game results are shown immediately after each round, allowing users to quickly see the winner and continue playing.

Score Tracking

Keeps track of player and computer scores to make the gameplay more engaging and competitive.
