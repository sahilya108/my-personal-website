Project Overview

Currency Converter is a web-based application developed using HTML, CSS, and JavaScript that allows users to convert amounts from one currency to another in real time.

The application fetches exchange rates and calculates the converted value instantly. This project focuses on understanding API integration, data handling, and creating a user-friendly financial utility tool.

Key Features

Real-Time Currency Conversion

Converts one currency to another instantly using the latest exchange rates for accurate results.

Multiple Currency Support

Supports a wide range of international currencies, allowing users to convert between different countries easily.

Instant Calculation

Displays converted values immediately as soon as the user enters the amount and selects currencies.

Easy Currency Selection

Provides simple dropdown menus to choose source and target currencies for smooth interaction.
